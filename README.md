# 2025_Spring_EmbededSystem
# Embedded Systems Programming with Mbed OS, Linux, and STM32

## 📚 Course Overview

This course is designed for junior and senior undergraduate students to develop a solid foundation in **embedded systems**, focusing on **Mbed OS (RTOS)**, **Linux-based development**, and **embedded C/C++ programming**. Students will learn how to integrate embedded **hardware**, **software**, and **middleware (OS libraries)** to implement real-world embedded applications.

---

## 🎯 Course Outcomes

By the end of this course, students will be able to:

- Understand and apply **RTOS principles** using Mbed OS
- Program with **C/C++** in embedded environments
- Integrate **sensors**, **Wi-Fi**, and **BLE modules**
- Develop on **STM32 platforms** using STM32CubeIDE
- Interface embedded systems with **Python clients (RPi)**
- Analyze embedded signals using **PWM**, **DMA**, and **logic analyzers**
- Implement **CMSIS-DSP** algorithms

---

## 🧪 Lab Exercises

| Lab | Description |
|-----|-------------|
| **Lab 1** | LED Blinking & RTOS API Introduction |
| **Lab 2** | Wi-Fi Connection and Sensor Data Handling |
| **Lab 3** | BLE Central (GATT Client) Programming |
| **Lab 4** | BLE Peripheral on STM32 (GATT Server) + Python Central on Raspberry Pi |
| **Lab 5** | PWM Signal Generation and Logic Analyzer Usage |
| **Lab 6** | Data Acquisition with DMA and Interrupt Handling |
| **Lab 7** | DSP Programming using CMSIS-DSP Libraries |
